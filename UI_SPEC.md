## 表示モード

### 小説モード (Novel)
- AI出力をそのままMarkdownレンダリング
- ユーザー入力は「主人公の行動」バッジ付き
- 最大幅800px、中央揃え

### チャットモード (Chat) — 推奨
AI出力を以下のセグメントに分解して表示:

#### 1. キャラクター吹き出し (左寄せ)
```
[アバター] キャラ名
           ┌──────────────────┐
           │「台詞テキスト」    │
           │                  │
           │ *動作/描写*       │
           │                  │
           │「続きの台詞」     │
           └──────────────────┘
```

#### 2. プレイヤー吹き出し (右寄せ)
```
                          主人公名 [アバター]
┌──────────────────┐
│ メッセージ内容    │  ← 紫色背景
└──────────────────┘
```

#### 3. ナレーションブロック (中央)
```
  ┃ 地の文テキスト。状況描写や心理描写など。
  ┃ キャラクターの発話を含まない部分。
```

## AI出力パーサー仕様

### パース対象パターン

#### キャラクター名ヘッダー
```
中野四葉        ← 単独行の短いテキスト（30文字以下）
「おはよう！」    ← 次の行が「で始まる場合、前行をスピーカーとして認識
```

#### インラインダイアログ
```
四葉「おはよう！」  ← 名前「台詞 のパターンを自動検出
```

#### 動作描写
```
*全力で駆け寄る*    ← アスタリスクで囲まれた行
＊顔をしかめる＊    ← 全角アスタリスクも対応
```

#### ナレーション
上記パターンに該当しない行はすべて地の文として扱う。

### キャラクターマッチ
1. 完全一致: `中野四葉` = `中野四葉`
2. 部分一致: `四葉` ⊂ `中野四葉` → マッチ
3. 未登録: デフォルトシルエットアバターを使用

## ヘッダーボタン

| ボタン | アイコン | 機能 |
|--------|----------|------|
| ≡ メニュー | `menu` | ストーリー一覧ドロワーを開く（モバイルのみ） |
| ストーリー | `menu_book` | ストーリーボード画面へ切替 |
| キャラクター | `group` | キャラクターライブラリ画面へ切替 |
| 設定 | `settings` | 設定画面へ切替 |
| 表示切替 | `forum`/`menu_book` | チャット/小説モード切替 |
| サイドバー | `right_panel_open` | 右サイドバードロワーを開く（モバイルのみ） |

## レスポンシブ・ブレークポイント

| 幅 | レイアウト |
|----|-----------|
| ≥ 1024px | PC: 左ドロワー常時表示、右サイドバー常時表示 |
| < 1024px | モバイル: 両ドロワーはスライドイン、チャットフルスクリーン |
