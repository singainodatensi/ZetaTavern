## 完了済みタスク

### v0.7 - チャットUI最適化 & 大画面トリミング & タグ絞り込み & 同期安全ガード (2026-05-23)
- [x] **チャット画面の地の文（ナレーション）のレイアウト・基準ライン同期**
  - 地の文のDOM構造を会話吹き出しと同じ `chat-message` flexレイアウト（非表示アバター枠＋テキスト枠）へ統合 [7.1]。
  - スマホ（48px）およびPC（62px）のアバター幅をスペーサーとして自動同期させ、セリフと地の文の左端開始ラインをピクセル単位で完全一致化 [4.1, 7.1]。
- [x] **Dropbox自動同期の安全ガード（F5先祖返り・データ消失バグの修正）**
  - 起動時（F5リロード）やタブ復帰時、ローカル側に最新の未同期変更（`timestamp` で判定）がある場合は、Pull（上書きクリア）をキャンセルし、自動でサイレントプッシュ（自動退避アップロード）を行う機能を追加 [7.1, 7.2]。
  - 重複実行ガード（`isSyncInProgress`）を導入し、Visibility APIと起動処理の並行競合によるDropboxの自己ロック（「他の端末が同期中です」エラー）を完全に解消 [7.1, 7.2]。
- [x] **キャラクター編集・ストーリー設定保存時のメモリState同期**
  - キャラクター追加/編集や主人公プロファイル編集の成功時に、メモリ上のグローバルState（`state.stories` / `state.characters`）へ即時反映し、リロードなしでUIが最新データに追従するよう改善 [7.1]。
- [x] **世界観を守るタグベースのキャラクター自動絞り込み**
  - ストーリーモデルとキャラクターモデルにタグ（`tags`）配列を拡張 [7.1]。
  - アクティブストーリーにタグが設定されている場合、サイドバーの登場キャラクター一覧にタグ一致キャラクターのみを自動でフィルタリング表示し、別作品キャラの混在を完全に防止（タグ未設定時は全表示のフォールバック） [7.1]。
  - キャラクターモーダルへのタグ入力欄（カンマ区切り）のJSによる動的インジェクション、およびライブラリでの「タグ一致」フィルターを実装 [7.1]。
- [x] **自由度を高めたフォントサイズ数値変更 (px指定) 機能**
  - 設定画面のプルダウン選択から、ユーザーが直接文字の大きさを数値入力（10px〜32px）できるインプット形式へ変更 [7.1]。
  - `--chat-font-size` などのCSS変数（`:root`）へ動的に注入し、ページリロードなしで会話・地の文・UI全体のフォントが同期して拡大縮小する設計へ刷新 [7.1]。
- [x] **地の文（ナレーション）の配色・透明度カスタマイズ**
  - 設定画面に地の文用の「背景のベース色（カラーパレット）」「文字のベース色」「不透明度スライダー」を追加 [7.1]。
  - 選択された配色をCSS変数（`--narration-bg` / `--narration-text`）経由で一括管理し、ユーザーごとの最適なコントラストに即時適用可能に改善 [7.1]。
- [x] **大画面トリミング調整モーダル（Crop Modal）の新設**
  - 小さいアイコン枠内での窮屈な直接編集を廃止し、画像アップロード時に快適な「大画面別枠トリミングモーダル」がポップアップする設計に移行 [4.1, 7.1]。
  - 円形の透過ガイドマスクを重ね、スマホのピンチズームやスライダー操作で直感的に丸枠内に顔を合わせられるUIを構築 [4.1, 7.1]。
  - プレビュー時のCSSアスペクト比を動的に算出・リサイズし、画面の見た目とCanvas切り出し範囲をピクセル単位で100%一致化 [7.1]。
  - セッション一時変数（`currentOriginalFile`）の保持により、再アップロードなしで同じ画像から「位置の再調整」がその場で何度でも可能な構造に拡張 [7.1]。
